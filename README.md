# Лента комментариев

_Учебный проект по frontend-разработке_

Одностраничное веб-приложение для публикации комментариев с авторизацией пользователей, лайками и возможностью отвечать на комментарии. Проект реализован на чистом JavaScript без использования фреймворков.

## Демо

https://kors-ak.github.io/webdev-dom-homework/

## Возможности

* Просмотр списка комментариев
* Добавление новых комментариев
* Авторизация пользователей
* Регистрация новых пользователей
* Лайки комментариев
* Ответ на комментарий по клику
* Валидация пользовательского ввода
* Защита от HTML-инъекций
* Обработка ошибок сети и сервера
* Сохранение данных пользователя в LocalStorage

## Технологии

* JavaScript (ES6 Modules)
* HTML5
* CSS3
* Fetch API
* REST API
* LocalStorage
* Git

## Основной функционал

### Авторизация и регистрация

Пользователь может создать аккаунт или войти в существующий. После успешной авторизации становится доступна форма публикации комментариев.

### Работа с комментариями

* получение комментариев с сервера;
* публикация новых комментариев;
* обновление списка после отправки;
* ответы на существующие комментарии.

### Лайки

Для каждого комментария реализована возможность поставить или убрать лайк с визуальной анимацией.

## Запуск проекта

1. Клонировать репозиторий:

```bash
git clone https://github.com/kors-ak/webdev-dom-homework
```

2. Перейти в папку проекта:

```bash
cd webdev-dom-homework
```

3. Открыть `index.html` в браузере или запустить локальный сервер.
